## Aim
To develop a prompt-based application using AI tools (ChatGPT, Claude, DeepSeek for text assistance; Runway for video generation; and Midjourney for image generation) to demonstrate how prompt engineering can be applied to organize daily tasks, create multimedia content, and provide adaptive, intelligent assistance.

---

## AI Tools Required
- Text Assistance: ChatGPT, Claude, DeepSeek  
- Video Generation: Runway  
- Image Generation: Midjourney  

---

## Explanation
This experiment focuses on creating a prompt-based personal productivity assistant using multiple AI tools and large language models (LLMs).  

The assistant can:
- Manage and organize daily tasks  
- Schedule activities and set reminders using natural language  
- Provide personalized wellness tips  
- Generate multimedia outputs (images and videos) based on prompts  

The goal is to understand how prompt engineering enables LLMs and generative AI tools to collaborate and solve real-world productivity and creativity challenges.
